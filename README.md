# BetterSoundMeeter

**BetterSoundMeeter** is a soundboard tool designed to enhance the Voicemeeter interface, specifically aimed at improving the functionality of **MacroButton**. This tool provides users with a more intuitive and customizable interface for controlling sound effects, managing audio levels, and automating tasks, making it easier to use Voicemeeter for live streaming, podcasting, or other audio-related applications.

## Features

- **Customizable Soundboard**: Easily trigger sound effects with customizable buttons.
- **MacroButton Integration**: Seamlessly integrate with Voicemeeter's MacroButton to automate actions and manage audio controls.
- **Application-specific volume control**: Adjust volume levels for individual applications independently from the system volume.
- **User-friendly interface**: A simple, clean, and easy-to-use interface designed for quick access to sound effects and controls.
- **Cross-platform support**: Works on Windows, macOS, and Linux (may vary depending on the platform).
